JOI 왕국의 새로운 기차 노선

JOI 왕국에는 N개의 역이 있으며, 역은 1번부터 N번까지 번호가 매겨져 있습니다.
또한 JOI 왕국에는 M개의 기차 노선이 있으며, 1번부터 M번까지 번호가 매겨져 있습니다.

각 기차 노선 i (1 ≤ i ≤ M)는 역 Ai와 역 Bi를 양방향으로 연결하며, 이동 시간은 Ci분이 걸립니다.

당신은 JOI 왕국의 장관으로서 새로운 기차 노선 하나를 건설하려고 합니다. 이 새로운 노선은 다음 조건을 만족해야 합니다:

  • 두 정수 u, v (1 ≤ u < v ≤ N)를 선택하여, 역 u와 v를 양방향으로 연결하는 노선을 추가합니다.
  • 이동 시간은 L분입니다.
  • 이미 u와 v를 잇는 노선이 존재하더라도 새로운 노선을 추가할 수 있습니다.

기차 노선이 새롭게 추가된 이후, 왕이 기뻐하는 조건은 다음과 같습니다:

역 S번에서 역 T번까지 총 이동 시간 K분 이하로 도달할 수 있다면 왕은 기뻐합니다.
(기차 대기 시간이나 환승 시간은 무시합니다.)

가능한 (u, v) 쌍은 총 N(N - 1) / 2가지가 존재합니다.
이 중 왕을 기쁘게 만들 수 있는 쌍의 개수를 구하세요.

입력

  • 정수 N, M (역 개수, 기존 노선 수)
  • M개의 줄: Ai, Bi, Ci (각 노선의 연결된 역과 이동 시간)
  • 정수 S, T (출발역, 도착역)
  • 정수 K (최대 허용 시간)
  • 정수 L (새 노선의 이동 시간)

출력

왕이 기뻐할 수 있는 (u, v) 쌍의 개수를 출력하세요.

입력

입력은 다음 형식으로 주어진다.


N M
S T L K
A₁ B₁ C₁ 
A₂ B₂ C₂ 

...

A_m B_m C_m

출력

출력으로 한 줄을 출력하세요. 출력은 왕을 기쁘게 만들 수 있는 두 정수 쌍의 개수를 포함해야 합니다.

제약조건

  • 2 ≤ N ≤ 200,000
  • 1 ≤ M ≤ 200,000
  • 1 ≤ S < T ≤ N
  • 1 ≤ L ≤ 109
  • 1 ≤ K ≤ 1015
  • 1 ≤ Ai < Bi ≤ N (1 ≤ i ≤ M)
  • (Ai, Bi) ≠ (Aj, Bj) (1 ≤ i < j ≤ M)
  • 1 ≤ Ci ≤ 109 (1 ≤ i ≤ M)
  • 주어지는 값은 모두 정수입니다.
